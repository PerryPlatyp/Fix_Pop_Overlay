# Fix_Pop_Overlay
## Edits launcher.js so you can get rid of the annoying orange overlay when using the launcher

###### Prerequisites 
- Python (obviously)

###### How to run
1. Download the file
2. Go to the directory where the file was saved
3. Run main.py as sudo **OR ELSE**
4. It will edit the file and ask you to restart GNOME shell
5. Done
